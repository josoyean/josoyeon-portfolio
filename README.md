
## **  
Intor**

리액트 기반으로 타입스크립트로 제작한 포트폴리오입니다.

DB는 SQL로 자유롭게 사용하고 싶어서 슈퍼베이스를 사용을해서 페이지를 제작했습니다.

## **사용기술**

lang - react, styled-components , typesjavascript, react-query 

배포 - vercel.app

## **개발 기간**

2025.09 ~ 진행중

## **구현 기능**

\- 모노톤 기반 원페이지 레이아웃과 스크롤 연동 네비게이션을 적용해, 섹션별 정보 탐색과 현재 위치 파악을 직관적으로 구성

 - 프로젝트 카드 클릭 시 모달로 상세 내용을 제공하고, 인터뷰 아코디언으로 Q&A를 단계적으로 확인할 수 있도록 UX 구성

\- Supabase와 React Query를 연동해 Skills·Experience·Projects 등 콘텐츠를 DB에서 관리하고 동적으로 렌더링하도록 구현

---

#### 1) Supabase를 활용해 DB 및 API를 연동

Supabase와 React Query를 연동해 Skills·Experience·Projects 등 콘텐츠를 DB에서 관리하고 동적으로 렌더링하도록 구현

#### 2) 모노톤 기반 원페이지 레이아웃과 스크롤 연동 네비게이션을 적용

깔끔하고 튀지 않은 색상과 스크롤 통해서 상단 네비게이션 동일한 섹션에 있으면 내 위치를 볼수있게 설정했습니다.

#### 3) 인터뷰 아코디언으로 Q&A를 단계적으로 확인할 수 있도록 UX 구성

제 생각을 Q&A로 만들어서 저의 생각을 짧게라고 적어 궁금해 하시는 사람들만 답변을 보게 만들었습니다
